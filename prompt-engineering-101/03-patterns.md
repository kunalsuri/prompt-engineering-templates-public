# Common Prompt Patterns

## 1. Instruction-Response
- Clear instruction followed by the model's response
- Example: "Translate the following text to French: 'Hello, how are you?'"

## 2. Role-Playing
- Assigning a specific role to the AI
- Example: "You are a helpful writing assistant..."

## 3. Few-Shot Learning
- Providing examples of input-output pairs
- Helps establish the desired format and style

## 4. Chain of Thought
- Asking the model to think step by step
- Improves reasoning and accuracy

## 5. Template Filling
- Using placeholders in structured prompts
- Useful for consistent formatting
