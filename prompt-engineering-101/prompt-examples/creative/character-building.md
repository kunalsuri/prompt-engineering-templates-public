# Character Building Prompts

## Detailed Character Creation
```
Create a detailed character profile for a [genre] story. Include:

1. Basic Information:
   - Full name
   - Age
   - Physical appearance
   - Occupation
   - Background/History

2. Personality:
   - Key traits (3-5 adjectives)
   - Motivations
   - Fears/Flaws
   - Character arc potential

3. Relationships:
   - Family
   - Friends
   - Rivals/Enemies
   - Love interests (if any)

4. Character-specific details:
   - Unique mannerisms
   - Speech patterns
   - Personal items
   - Secrets

5. Role in story:
   - How they fit into the main plot
   - Character growth opportunities
   - Potential conflicts
```

## Character Interview
```
Conduct a character interview for [character name]. Ask 10 probing questions that reveal:
1. Their deepest fear
2. Greatest desire
3. Biggest regret
4. Best memory
5. How they handle conflict
6. What they value most
7. Their biggest secret
8. How they've changed over time
9. Their opinion of themselves vs how others see them
10. What they would change about their past

For each question, provide a detailed response in the character's voice.
```

## Character Development Exercise
```
Help me develop my character by exploring their backstory through these key life events:

1. Childhood:
   - Earliest memory
   - Family dynamics
   - Formative experience

2. Adolescence:
   - First major challenge
   - Important friendship/relationship
   - Moment that shaped their worldview

3. Adulthood:
   - Defining choice or event
   - Professional turning point
   - Current situation and goals

For each stage, provide:
- What happened
- How it affected them
- How it influences their current behavior
- Potential for future development

Character details: [brief description of character]
