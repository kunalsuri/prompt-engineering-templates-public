# Plot Generation Prompts

## Story Structure Development
```
Help me develop a [genre] story using the [three-act/five-act/hero's journey] structure.

Current story elements:
- Main character: [brief description]
- Setting: [brief description]
- Central conflict: [brief description]

For each act/phase, provide:
1. Key events
2. Character development points
3. Rising tension
4. Major turning points
5. Resolution elements

Additional notes: [any specific themes, tone, or elements to include]
```

## Plot Twist Generator
```
Generate 5 unexpected plot twists for a [genre] story with these elements:
- Main character: [description]
- Current situation: [brief summary]
- Antagonist/villain: [if applicable]

For each twist, include:
1. How it's set up
2. When it's revealed
3. Impact on characters and plot
4. How it resolves
5. Potential red herrings or misdirection

Tone: [serious/dark/humorous/etc.]
```

## World-Building Prompts
```
Help me build a detailed [fantasy/sci-fi/alternate history/etc.] world for my story.

Please develop the following aspects:
1. Geography and environment
2. Culture and society
3. Political systems
4. Magic/technology systems (if applicable)
5. History and lore
6. Daily life and customs
7. Economy and trade
8. Religion and belief systems

For each aspect, provide:
- Key features
- How it affects the story/characters
- Unique elements that set it apart
- Potential conflicts or tensions

Story themes: [list any specific themes to reflect in the world-building]
