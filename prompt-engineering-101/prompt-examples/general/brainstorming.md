# Brainstorming Prompts

## General Brainstorming
```
Brainstorm [number] creative ideas for [topic]. For each idea, provide:
1. A brief description
2. Potential benefits
3. Possible challenges
4. First steps to implement

Focus on [specific aspect or constraint if any].
```

## Product Development Brainstorm
```
You are a product development expert. Generate [number] innovative product ideas that:
- Solve [specific problem]
- Target [specific audience]
- Use [specific technology/approach, if any]

For each idea, include:
1. Product name
2. Core value proposition
3. Key features
4. Potential market size
5. Implementation complexity (1-5 scale)
```

## Content Ideas Generation
```
Generate [number] engaging content ideas about [topic] for [platform, e.g., blog, YouTube, podcast].

For each idea, provide:
1. Working title
2. Target audience
3. Key points to cover
4. Potential engagement hooks
5. Related keywords/tags

Tone: [specify tone, e.g., professional, casual, educational]
