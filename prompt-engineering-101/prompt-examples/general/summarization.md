# Summarization Prompts

## Basic Summarization
```
Please summarize the following text in 2-3 concise sentences. Focus on the main points and key information.

[Insert text to summarize here]
```

## Technical Paper Summary
```
You are an expert in [field]. Please provide a structured summary of this research paper that includes:
1. Main research question
2. Methodology
3. Key findings
4. Significance/Implications

[Insert paper text or abstract]
```

## Meeting Notes Summary
```
Please summarize these meeting notes, highlighting:
- Key decisions made
- Action items (who is responsible for what)
- Important discussion points
- Next steps with deadlines

[Insert meeting notes]
