# Translation Prompts

## Basic Translation
```
Translate the following text from [source language] to [target language]. Maintain the original tone and style:

[Insert text to translate]
```

## Localization
```
Localize the following content for [target region/country]. Consider cultural references, idioms, and local expressions:

[Insert content to localize]

Additional context:
- Target audience: [describe]
- Purpose: [e.g., marketing, technical documentation, casual conversation]
- Tone: [specify]
```

## Technical Translation
```
Translate the following technical documentation from [source language] to [target language]. Ensure technical accuracy and consistency with standard terminology in the field of [specific field].

[Insert technical content]

Please:
1. Maintain any code snippets or technical terms in their original form
2. Use standard terminology from [specific field]
3. Keep the same level of formality as the original
4. Preserve any markdown or formatting

For any ambiguous terms, please provide a brief note explaining your translation choice.
